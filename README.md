
# MEMENTO NOTES
Memento Notes is a note taking web app which allows its user to add , delete , edit notes etc. It is also provided with other features such as Archives, filter and sort notes.

## Tech Stack

React JS 

##Pages

*Landing Page
*Notes Page
*Archive Page
*Labels Page
*Sign Up Page
*Login Page
